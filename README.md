# Wimps Bundle Shop

Instant data bundle delivery for MTN, Telecel, and AirtelTigo in Ghana.  
Pay with Paystack - MoMo, Card, or Bank Transfer. Data lands in seconds.

**Built by: DEFCON 1 Dev Team**

## Features
- **Instant Delivery**: Bundles delivered automatically via VTU API
- **3 Networks**: MTN, Telecel, AirtelTigo all supported  
- **Paystack Only**: Secure payments with MoMo, Card, Bank Transfer
- **User Dashboard**: Transaction history + balance tracking
- **Referral System**: Earn when you refer friends

## Tech Stack
Next.js + Node.js + Sqlite + Paystack API + VTU Provider API
